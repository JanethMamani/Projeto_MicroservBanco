O aplicativo IndicadoresBanco possui as classes:

ExpectativaServico responsável por separar os valores do arquivo Json.
ExpectativaRecursos responsável por requisitar os endpoints
Além de demais como a aplication main e a objeto para conter a estrutura principal.

O objeto Json deve ser colocado na pasta do aplication, para que o JSONObject seja gerado a partir dele, com {} sendo os primeiros carcteres.
